# Select

This components extends the [react-native-modal-dropdown](https://www.npmjs.com/package/react-native-modal-dropdown).

### Usage

Simple Example:
```
<Select
  defaultIndex={1}
  options={[1, 2, 3, 4, 5]}
  style={styles.shadow}
/>
```
